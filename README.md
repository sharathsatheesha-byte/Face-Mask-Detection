# Face Mask Detection System

A deep learning–based face mask detection system using Convolutional Neural Networks (CNN).
The project is deployed as a Flask web application for real-time image prediction.

## Features
- Image-based face mask detection
- CNN model for classification
- Flask web interface
- Real-time prediction

## Technologies Used
- Python
- TensorFlow & Keras
- Convolutional Neural Networks (CNN)
- Flask
- HTML, CSS

## How to Run

1. Install dependencies
```bash
pip install -r requirements.txt
```

2. Run the application
```bash
python app.py
```

3. Open your browser and visit:
```
http://127.0.0.1:5000/
```
